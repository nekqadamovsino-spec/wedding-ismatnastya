Сайт Ismat & Nastya

1. Музыка: положите MP3-файл в assets/music.mp3.
2. Анкета Google Sheets:
   - Создайте таблицу Google Sheets.
   - Расширения → Apps Script.
   - Вставьте код из apps-script.gs.
   - Развернуть → Новое развертывание → Веб-приложение.
   - Доступ: Все.
   - Скопируйте URL и вставьте его в script.js вместо SCRIPT_URL=''.
3. Загрузите все файлы в корень репозитория GitHub Pages.
